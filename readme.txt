08番 菊地巧椰(220065)

ブロックの上げ下げ、ブロックの種類を変更、カメラの移動
遠藤先生の完成版サンプルに追加されてた機能はすべて追加しましたが、レイの当たり判定だけ少しずれていて解決できませんでした。

15×15のブロックのどこからクリックしても全てのdistを比べてちゃんと一番近い位置のレイが当たった所のブロックを処理するようにした。
一応スペースで上に上がりシフトで下に下がるようにしましたが、遠くへ行くとおそらく当たり判定が距離に応じて小さくなっていないので判定が変になりました。

モデルの判定やレイの事はある程度わかりましたが、逆行列や変換行列がまだ完全には理解しきれませんでした。
また、レイを飛ばした時の当たり判定が少しずれているのですが、それの解決法がわかりませんでした。
チーム制作で作ったマップエディターに簡易的な予測設置位置のような処理を追加したいですが、他の部分の開発も並行して行っている
為その部分が終わったら自分のマップエディターに生かしたいです。
